#  Simple Task Manager (Python)

This is a basic command-line task manager made using Python.
## Features

* Add new tasks
* View all tasks
* Mark tasks as done
* Tasks are saved in a file (tasks.txt)

## How to Run

1. Make sure Python is installed
2. Run the file:

```bash
python your_file_name.py
```

# File Used

* tasks.txt → stores all tasks

## Example

```
1. add task
2. view tasks
3. mark task as done
4. exit
```

##  Purpose

This project is made for learning basic Python concepts like:

* loops
* conditions
* file handling
* dictionaries
